# Qtile configuration

Configuración básica para usar qtile luego de haberlo instalado

Para saber los comandos básicos revisar el archivo: settings/keys.py


para poder usarse se deben instalar los siguientes programas:

    $ pacman -S brave
    $ pacman -S rofi
    $ pacman -S thunar
    $ pacman -S alacritty
    $ pacman -S redshift
